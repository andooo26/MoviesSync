# MoviesSync
- 2025学園祭のさざなみ開発ブースのコーナー「モニター映像展示」で使用するAndroid向けアプリ

## 機能
- 同期再生:複数デバイスで同時に動画を再生
- グループ機能:ホストが建てるグループに対してクライアントで参加を行う

## アプリの使用方法
### ホスト側の設定
1. アプリを立ち上げる
2. 「グループを作成」を押下
3. 「動画を選択」より再生する動画を選択する
4. クライアントが参加し終えたら「再生」を押下

### クライアント側の設定
1. アプリを立ち上げる
2. 「QRコードで参加」を押下
3. ホスト側のQRコードを読み込む
4. 「動画を選択」より再生する動画を選択する

## 技術仕様
- 言語:Java
- 最小SDK:26(Android 8.0)
- 主要パッケージ:
  - ExoPlayer
  - ZXing Android Embedded

## NW周り
- ホスト側でServerSocketでポート8888をリッスン
- クライアントのIPへ接続
- メッセージプロトコル
  - CONNECT,           // クライアント接続通知
  - CONNECTED,         // 接続確認応答
  - VIDEO_METADATA,    // 動画情報
  - VIDEO_DATA,        // 動画データチャンク
  - READY,            // 受信完了通知
  - SYNC_TIME,        // 時刻同期
  - PLAY_COMMAND,     // 再生開始指示
  - PAUSE_COMMAND,    // 一時停止指示
  - LOOP_END,         // 再生終了通知（次ループ用）
  - ERROR             // エラー

### Q&A
- グループへの参加に失敗する
    - 同一NWにいるかを確認してください
    - アプリを再起動してください

- QRコードが表示されない
    - Wi-Fiが接続されているかを確認してください

- アプリが応答しない
    - アプリを再起動してください
    
- 遅延する
